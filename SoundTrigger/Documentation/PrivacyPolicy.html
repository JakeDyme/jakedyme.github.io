Summary

Policy

Types of Data generated :
* This application records audio data using the device’s microphone. All recordings are kept within the application’s domain, on the user’s device. All recorded data acquired by the application, is removed upon deletion of the application, under the governance of the operating system. This application does not send any personal information or recorded data over the internet.

Processing Details

Main Application

While the application is active, and after the user has pressed the record button, the application will record one second of sound-data, using the microphone, each second until the user presses the stop button. Each recorded sample of one second is analyzed for sounds and then immediately deleted.

Background Service

A background service is activated when the user navigates away from the application. This is to ensure that the application continues to monitor for, and respond to, sounds even when the screen goes into sleep mode. The primary use-case of this application is to respond to sounds, whilst remaining unattended. This requires the application to continue as a background service. The service is visible in the user’s notification menu and can be monitored and controlled from the notification menu.

Application Communication

The main application will communicate with the internet for the following reasons:
  1. To retrieve advertisment data from the Google advertisment provider AdMob upon loading of the application.
  2. To retrieve this policy from the application's website, should the user click on the "Privacy Policy" menu item.
  3. To retrieve help information from the application's website.
  4. To submit statistical counter information to Google AdMob, should the user click on any advertisments.
If there are any communications outside of this list, they should not be considered an intentional design aspect of this application, and may be the result of sealed third-party libraries used by this applciation. Should any such communications be detected, and be of a security concern, it is acceptable to bring it to the attention of the developer, who shall make a best effort to investigate.
